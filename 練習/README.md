# Mini Games and Apps

## ゲーム一覧

### テトリス (Tetris)
クラシックなテトリスゲームをHTML5 Canvasで実装。

**ファイル**: `tetris.html`

**操作方法**:
| キー | アクション |
|-----|-----------|
| ← → | 左右移動 |
| ↑ | 回転 |
| ↓ | 高速落下 |
| Space | 即落下 |
| P | 一時停止 |

**特徴**:
- 7種類のテトリミノ
- ゴースト表示（落下予測位置）
- レベルシステム（10ライン毎にレベルアップ）
- スコアリング（クラシックテトリス方式）
- 次のブロックプレビュー
